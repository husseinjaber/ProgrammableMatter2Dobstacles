Interface Help

Keyboard:
      [SPACE]: pause/resume the simulation (REALTIME mode only)
          [z]: center camera focus on the selected module
      [w]/[W]: toggle full screen/window mode
      [r]/[R]: start in realtime/fastest mode
          [h]: show/hide help window
          [i]: show/enlarge/hide console window
          [s]: save a screenshot to the app folder
          [S]: starts/ends a video capture to the app folder
          [B]: toggle background
          [!]: export an STL model of the configuration
          [,]: show FPS
        [1-9]: set the color of the selected module
      [f]/[F]: toggle OpenGL line/fill drawing mode
      [+]/[-]: zoom in/out
      [T]/[t]: toggle large/small console font
[PgUp]/[PgDw]: Increase/Decrease module motion speed
[q]/[Q]/[ESC]: quit

Mouse:
        <MLB>: change the point of view, tap on a selected module
        <MRB>: change the focus point
 [Ctrl]+<MLB>: select a module
 [Ctrl]+<MRB>: context menu
            {Add module} add a module connected to the selected face
            {Delete module} delete the selected module
            {Tap} schedule a tap event for the selected module
            {Save} save the current configuration in the config file.
            {Cancel} close the menu
